# Курсовая: Управление пациентами и ЭМК (Вариант 49)

## Стек
- Backend: Java 17, Spring Boot 2.7
- Frontend: HTML + CSS + Bootstrap + JS (шаблонизация на сервере)
- БД: PostgreSQL 14

## Модули

### Регистратура
- Регистрация пациента: ФИО, пол, дата рождения, контакты, паспортные данные
- Уникальный номер истории болезни (УИН)
- Прикрепление/открепление к терапевтическому участку
- Учёт льготных категорий и спецотметок

### ЭМК
- Ведение электронной медицинской карты
- Архив карт выбывших пациентов
- Журнал аудита изменений

### Отчётность
- Отчёты по нагрузке персонала
- Формирование и печать статистического талона
- Фильтрация по периоду

## Нефункциональные требования
- Поиск пациента < 3 сек, загрузка страницы < 2 сек
- Авторизация по паролю (хеширование)
- Весь трафик TLS 1.2+
- Автобэкап, логирование ошибок
- Поддержка печати на принтере

## Вариант (формулировка)

Больница.Управление пациентами и медицинскими картами
1. Регистрация нового пациента: ФИО, пол, дата рождения, контакты, паспортные данные, полис ОМС/ДМС
2. Ведение электронной медицинской карты (ЭМК) пациента (CRUD, фильтр и поиск)
3. Создание и ведение единого номера истории болезни (УИН)
4. Прикрепление и открепление пациента к терапевтическому участку
5. Учет льготных категорий пациентов и специальных отметок (инвалидность, аллергии)
6. Ведение архива медицинских карт выбывших пациентов
7. Контроль целостности и неизменности данных ЭМК (ведение журнала изменений)
8. Формирование и печать статистического талона и других медицинских форм
